20class STS_epr(): 

21 

22 """STS-EPR model. 

23  

24 The STS-EPR (Spatial, Temporal and Social EPR model) model of individual human mobility consists of the following mechanisms [CRP2020]_:  

25  

26  

27 **Action selection**. With probability :math:`P_{exp}=\\rho S^{-\\gamma}`, where :math:`S` is the number of distinct locations previously visited by the agent,  

28 the agent visits a new location (Exploration), otherwise with a complementary probability :math:`P_{ret}=1-P{exp}` it returns to a previously visited location (Return).  

29 At that point, the agent determines whether or not the location’s choice will be affected by the other agents; with a probability :math:`\\alpha`, the agent’s social contacts influence its 

30 movement (Social). With a complementary probability of :math:`1-\\alpha`, the agent’s choice is not influenced by the other agents (Individual). 

31 

32 Parameters :math:`\\rho`, :math:`\\gamma`, and :math:`\\alpha=` correspond to arguments `rho`, `gamma`, and `alpha` of the constructor, respectively. 

33  

34 After the selection of the spatial mechanism (Exploration or Return) and the social mechanism (Individual or Social)  

35 decides which location will be the destination of its next displacement during the **Location selection phase**. 

36 For an agent :math:`a`, we denote the sets containing the indices of the locations :math:`a` can explore or return, as :math:`exp_{a}` and :math:`ret_{a}`, respectively. 

37 

38 **Individual Exploration**. If the agent :math:`a` is currently in location :math:`i`, and explores a new location without the influence of its social contacts, then the new location :math:`j \\neq i` is an unvisited location for the agent (:math:`i \\in exp_{a}`)  

39 and it is selected according to the gravity model with probability proportional to :math:`p_{ij} = \\frac{r_i r_j}{dist_{ij}^2}`, where :math:`r_{i (j)}` is the location's relevance, that is, the probability of a population to visit location :math:`i(j)`, :math:`dist_{ij}` is the geographic distance between :math:`i` and :math:`j`,  

40 The number of distinct locations visited, :math:`S`, is increased by 1. 

41 

42 **Social Exploration**. If the agent :math:`a` is currently in location :math:`i`, and explores a new location with the influence of a social contact, it first selects a social contact :math:`c`  

43 with probability :math:`p(c) \\propto mob_{sim}(a,c)` [THSG2015]_. At this point, the agent :math:`a` explores an unvisited location for agent :math:`a` that was visited by agent :math:`c`, i.e., the location :math:`j \\neq i` is selected 

44 from set :math:`A = exp_a \\cap ret_c`; the probability :math:`p(j)` for a location :math:`j \\in A`, to be selected is proportional to :math:`\Pi_j = f_j`, where :math:`f_j` is the visitation frequency of location :math:`j` for the agent :math:`c`. The number of distinct locations visited, :math:`S`, is increased by 1.  

45 

46 **Individual Return**. If the agent :math:`a`, currently at location :math:`i`, returns to a previously visited location :math:`j \\in ret_a`, it is chosen with probability  

47 proportional to the number of time the agent visited :math:`j`, i.e., :math:`\Pi_j = f_j`, where :math:`f_j` is the visitation frequency of location :math:`j`. 

48 

49 **Social Return**. If the agent :math:`a` is currently in location :math:`i`, and returns to a previously visited location with the influence of a social contact, it first selects a social contact :math:`c`  

50 with probability :math:`p(c) \\propto mob_{sim}(a,c)` [THSG2015]_. At this point, the agent :math:`a` returns to a previously visited location for agent :math:`a` that was visited by agent :math:`c` too, i.e., the location :math:`j \\neq i` is selected 

51 from set :math:`A = ret_a \\cap ret_c`; the probability :math:`p(j)` for a location :math:`j \\in A`, to be selected is proportional to :math:`\Pi_j = f_j`, where :math:`f_j` is the visitation frequency of location :math:`j` for the agent :math:`c`. 

52 

53  

54  

55 parameters 

56 ---------- 

57 name : str, optional 

58 the name of the instantiation of the STS-EPR model. The default value is "STS-EPR". 

59 rho : float, optional 

60 it corresponds to the parameter :math:`\\rho \in (0, 1]` in the Action selection mechanism :math:`P_{exp} = \\rho S^{-\gamma}` and controls the agent's tendency to explore a new location during the next move versus returning to a previously visited location. The default value is :math:`\\rho = 0.6` [SKWB2010]_. 

61 gamma : float, optional 

62 it corresponds to the parameter :math:`\gamma` (:math:`\gamma \geq 0`) in the Action selection mechanism :math:`P_{exp} = \\rho S^{-\gamma}` and controls the agent's tendency to explore a new location during the next move versus returning to a previously visited location. The default value is :math:`\gamma=0.21` [SKWB2010]_. 

63 alpha : float, optional 

64 it corresponds to the parameter `\\alpha` in the Action selection mechanism and controls the influence of the social contacts for an agent during its location selection phase. The default value is :math:`\\alpha=0.2` [THSG2015]_.

130 def make_social_action(self, agent, mode): 

131 

132 '''  

133 The agent A makes a social choice in the following way: 

134 

135 1. The agent A selects a social contact C with probability proportional to the  

136 mobility similarity between them 

137  

138 2. The candidate location to visit or explore is selected from the set composed of  

139 the locations visited by C (ret(C)), that are feasible according to A's action:  

140 - exploration: exp(A) \intersect ret(C) 

141 - return: ret(A) \intersect ret(C) 

142  

143 3. select one of the feasible locations (if any) with a probability proportional 

144 to C's visitation frequency 

145 ''' 

146 

147 contact_sim = [] 

148 

149 #check and update the mobility similarity of the agent's edges if 'expired' 

150 for ns in self.social_graph.neighbors(agent): 

151 eid = self.social_graph.get_eid(agent,ns) 

152 

153 if self.social_graph.es(eid)['next_update'][0] <= self.current_date: 

154 #update 

155 lv1 = self.agents[agent]['location_vector'] 

156 lv2 = self.agents[ns]['location_vector'] 

157 self.social_graph.es(eid)['mobility_similarity'] = self.cosine_similarity(lv1,lv2) 

158 self.social_graph.es(eid)['next_update'] = self.current_date + datetime.timedelta(hours=self.dt_update_mobSim) 

159 

160 contact_sim.append(self.social_graph.es(eid)['mobility_similarity'][0]) 

161 

162 contact_sim = numpy.array(contact_sim) 

163 

164 if len(contact_sim)!=0: 

165 if numpy.sum(contact_sim)!=0: 

166 contact_pick = self.random_weighted_choice(contact_sim) 

167 else: 

168 contact_pick = numpy.random.randint(0, len(contact_sim)) 

169 

170 contact = [i for i in self.social_graph.neighbors(agent)][contact_pick] 

171 

172 else: 

173 #no contact in the social network, can not make a social choice 

174 return -1 

175 

176 # get the location vectors of the agent and contact 

177 location_vector_agent = self.agents[agent]['location_vector'] 

178 location_vector_contact = self.agents[contact]['location_vector'] 

179 

180 # id_locs_feasible, a vector of indices containing all the agent's feasible location (depend on the mode) 

181 if mode == 'exploration': 

182 id_locs_feasible = numpy.where(location_vector_agent==0)[0] 

183 if mode == 'return': 

184 id_locs_feasible = numpy.where(location_vector_agent>=1)[0] 

185 

186 # the constraint set is of the form {current_location, starting_location} 

187 id_locs_constrain_diary = [self.agents[agent]['current_location']]+[self.agents[agent]['home_location']] 

188 id_locs_feasible = [loc_id for loc_id in id_locs_feasible if loc_id not in id_locs_constrain_diary] 

189 

190 #no location selectable for the agent in the current mode 

191 if len(id_locs_feasible) == 0: 

192 return -1 

193 

194 id_locs_valid = id_locs_feasible 

195 

196 #project v_location with the indices in id_locs_valid 

197 v_location_proj = [location_vector_contact[i] for i in id_locs_valid] 

198 

199 if numpy.sum(v_location_proj) != 0: 

200 #weighted choice 

201 idx = self.random_weighted_choice(v_location_proj) 

202 location_id = id_locs_valid[idx] 

203 else: 

204 location_id = -1 

205 

206 return location_id

243 def make_individual_exploration_action(self, agent): 

244 ''' 

245 The agent A, current at location i selects an UNVISITED location (i.e., in exp(A)) 

246 j with probability proportional to (r_i * r_j)/ d_ij^2 

247 

248 ''' 

249 

250 v_location = self.agents[agent]['location_vector'] 

251 

252 # compute the indices of all the feasible locations for the agent A (the unvisited ones) 

253 id_locs_feasible = numpy.where(v_location==0)[0] 

254 

255 id_locs_constrain_diary = [self.agents[agent]['current_location']]+[self.agents[agent]['home_location']] 

256 

257 id_locs_feasible = [loc_id for loc_id in id_locs_feasible if loc_id not in id_locs_constrain_diary ] 

258 

259 if len(id_locs_feasible) == 0: 

260 return -1 

261 

262 

263 src = self.agents[agent]['current_location'] 

264 self.compute_od_row(src) 

265 distance_row = numpy.array((self.distance_matrix[src].todense())[0])[0] 

266 id_locs_valid = id_locs_feasible 

267 

268 #this is made to avoid d/0  

269 distance_row[src]=1 

270 

271 relevance_src = self.relevances[src] 

272 distance_row_score = numpy.array(1/distance_row**2) 

273 distance_row_score = distance_row_score * self.relevances * relevance_src 

274 

275 #avoid self return 

276 distance_row[src]=0 

277 

278 v_location_proj = numpy.array([distance_row_score[i] for i in id_locs_valid]) 

279 

280 #weighted choice 

281 idx = self.random_weighted_choice(v_location_proj) 

282 location_id = id_locs_valid[idx] 

283 

284 return location_id

491 def distance_earth_km(self, src, dest): 

492 

493 lat1, lat2 = src['lat']*pi/180, dest['lat']*pi/180 

494 lon1, lon2 = src['lon']*pi/180, dest['lon']*pi/180 

495 dlat, dlon = lat1-lat2, lon1-lon2 

496 

497 ds = 2 * asin(sqrt(sin(dlat/2.0) ** 2 + cos(lat1) * cos(lat2) * sin(dlon/2.0) ** 2)) 

498 return 6371.01 * ds

533 delta_T = self.agents[agent]['time_next_move']-self.current_date 

534 dT = delta_T.components[0]*24 + delta_T.components[1] 

535 next_move = str(self.agents[agent]['time_next_move']) 

536 else: 

537 self.agents[agent]['time_next_move'] = self.end_date + datetime.timedelta(hours=1) 

538 dT = 1 

539 next_move = "None" 

540 

541 self.agents[agent]['dt'] = dT 

542 self.store_tmp_movement(self.current_date, agent, location_id, dT) 

543 

544 return {'from':from_, 'to': location_id, 'next_move':next_move}

549 

550 '''  

551 The implementation of the action-correction phase, executed by an agent if 

552 the location selection phase does not allow movements in any location  

553 ''' 

554 corrections=[] 

555 

556 if choice == 'social_return': 

557 location_id = self.make_individual_return_action(agent) 

558 corrections.append('individual_return') 

559 if location_id < 0: 

560 choice = 'individual_return' 

561 

562 elif choice == 'social_exploration': 

563 location_id = self.make_individual_exploration_action(agent) 

564 corrections.append('individual_exploration') 

565 if location_id < 0: 

566 choice = 'individual_exploration' 

567 

568 if choice == 'individual_return': 

569 location_id = self.make_individual_exploration_action(agent) 

570 corrections.append('individual_exploration') 

571 

572 elif choice == 'individual_exploration': 

573 location_id = self.make_individual_return_action(agent) 

574 corrections.append('individual_return') 

575 

576 return location_id, corrections

642 def generate(self, start_date, end_date, spatial_tessellation, diary_generator, 

643 social_graph='random', n_agents=500, rsl=False, distance_matrix=None, 

644 relevance_column=None, min_relevance = 0.1, dt_update_mobSim = 24*7, 

645 indipendency_window = 0.5, random_state=None, log_file=None, verbose=0, 

646 show_progress=False): 

647 

648 """ 

649 Start the simulation of a set of agents at time `start_date` till time `end_date`. 

650  

651 Parameters 

652 ---------- 

653 start_date : datetime 

654 the starting date of the simulation, in "YYY/mm/dd HH:MM:SS" format. 

655 end_date : datetime 

656 the ending date of the simulation, in "YYY/mm/dd HH:MM:SS" format. 

657 spatial_tessellation : pandas DataFrame or geopandas GeoDataFrame 

658 the spatial tessellation, i.e., a division of the territory in locations. 

659 diary_generator : MarkovDiaryGenerator 

660 the diary generator to use for generating the mobility diary [PS2018]_. 

661 social_graph : "random" or an edge list 

662 the social graph describing the sociality of the agents. The default is "random".  

663 n_agents : int, optional 

664 the number of agents to generate. If `social_graph` is "random", `n_agents` are initialized and connected, otherwise the number of agents is inferred from the edge list. The default is 500.  

665 rsl: bool, optional 

666 if Truen the probability :math:`p(i)` for an agent of being assigned to a starting physical location :math:`i` is proportional to the relevance of location :math:`i`; otherwise, if False, it is selected uniformly at random. The defailt is False. 

667 distance_matrix: numpy array or None, optional 

668 the origin destination matrix to use for deciding the movements of the agent. If None, it is computed “on the fly” during the simulation. The default is None. 

669 relevance_column: str, optional 

670 the name of the column in spatial_tessellation to use as relevance variable. The default is “relevance”. 

671 min_relevance: float, optional 

672 the value in which to map the null relevance. The default is 0.1. 

673 random_state : int or None, optional 

674 if int, it is the seed used by the random number generator; if None, the random number generator is the RandomState instance used by np.random and random.random. The default is None.  

675 dt_update_mobSim: float, optional 

676 the time interval (in hours) that specifies how often to update the weights of the social graph. The default is 24*7=168 (one week). 

677 indipendency_window: float, optional 

678 the time window (in hours) that must elapse before an agent's movements can affect the movements of other agents in the simulation. The default is 0.5. 

679 log_file : str or None, optional 

680 the name of the file where to write a log of the execution of the model. The logfile will contain all decisions made by the model. The default is None. 

681 verbose: int, optional 

682 the verbosity level of the model relative to the standard output. If `verbose` is equal to 2 the initialization info and the decisions made by the model are printed, if `verbose` is equal to 1 only the initialization info are reported. The default is 0. 

683 show_progress : boolean, optional 

684 if True, show a progress bar. The default is False. 

685  

686 Returns 

687 ------- 

688 TrajDataFrame 

689 the synthetic trajectories generated by the model 

690 """ 

691 

692 

693 

694 # check arguments 

695 if n_agents<=0: 

696 raise ValueError("Argument 'n_agents' must be > 0.") 

697 if start_date > end_date : 

698 raise ValueError("Argument 'start_date' must be prior to 'end_date'.") 

699 if type(rsl) != bool: 

700 raise TypeError("Argument `rsl` must be a bool.") 

701 

702 

703 

704 # init data structures and parameters  

705 self.n_agents = n_agents 

706 self.tmp_upd = [] 

707 self.trajectories = [] 

708 self.dt_update_mobSim = dt_update_mobSim 

709 self.indipendency_window = indipendency_window 

710 self.verbose = verbose 

711 self.log_file = log_file 

712 

713 if rsl: 

714 self.starting_locations_mode = 'relevance' 

715 else: 

716 self.starting_locations_mode = 'uniform' 

717 

718 self.start_date, self.current_date, self.end_date = start_date, start_date, end_date 

719 

720 

721 # INITIALIZATION 

722 

723 #if specified, fix the random seeds to guarantee the reproducibility of the simulation 

724 if random_state is not None: 

725 numpy.random.seed(random_state) 

726 

727 #log_file 

728 if log_file is not None: 

729 self._log_file = log_file 

730 logging.basicConfig(format='%(message)s', filename=log_file, filemode='w', level=logging.INFO) 

731 

732 #Mobility diary generator 

733 if type(diary_generator) == MarkovDiaryGenerator: 

734 self.diary_generator = diary_generator 

735 else: 

736 raise TypeError("Argument `diary_generator` should be of type skmob.models.markov_diary_generator.MarkovDiaryGenerator.") 

737 

738 #time interval of the simulation  

739 delta_T = (self.end_date - self.start_date) 

740 self.total_h = delta_T.components[0]*24 + delta_T.components[1] 

741 #init. a progress bar with hourly precision 

742 if show_progress: 

743 last_t = self.start_date 

744 pbar = tqdm(total=self.total_h) 

745 elapsed_h = 0 

746 

747 #init. the spatial tessellation 

748 self.init_spatial_tessellation(spatial_tessellation, relevance_column, min_relevance) 

749 

750 #distance matrix  

751 if distance_matrix is not None: 

752 self.distance_matrix = distance_matrix 

753 print("Pre-computed matrix") 

754 else: 

755 self.distance_matrix = scipy.sparse.lil_matrix((len(self.spatial_tessellation),len(self.spatial_tessellation))) 

756 

757 

758 #init. the agents and social graph  

759 self.init_agents_and_graph(social_graph) 

760 

761 

762 #log init. info 

763 if self.log_file is not None: 

764 logging.info("model:\t"+self.name) 

765 logging.info("time interval:\t["+str(self.start_date)+" - "+str(self.end_date)+"]") 

766 logging.info("#agents:\t"+str(self.n_agents)) 

767 logging.info("#locations:\t"+str(len(self.spatial_tessellation))) 

768 logging.info("starting locations:\t"+self.starting_locations_mode) 

769 if self.map_ids: 

770 logging.info("social graph:\t argument") 

771 else: 

772 logging.info("social graph:\t random") 

773 logging.info("#edges:\t"+str(len(self.social_graph.es))) 

774 logging.info("random state:\t"+str(random_state)+"\n\n") 

775 

776 if self.verbose>0: 

777 print("Model:\t"+self.name) 

778 print("time interval:\t["+str(self.start_date)+" - "+str(self.end_date)+"]") 

779 print("#agents:\t"+str(self.n_agents)) 

780 print("#locations:\t"+str(len(self.spatial_tessellation))) 

781 print("starting locations:\t"+self.starting_locations_mode) 

782 if self.map_ids: 

783 print("social graph:\t argument") 

784 else: 

785 print("social graph:\t random") 

786 print("#edges:\t"+str(len(self.social_graph.es))) 

787 print("random state:\t"+str(random_state)+"\n\n") 

788 

789 

790 while self.current_date < self.end_date: 

791 

792 # we can update all the trajectories made OUTSIDE the indipendence window.  

793 sup_indipendency_win = self.current_date - datetime.timedelta(hours=self.indipendency_window) 

794 self.update_agent_movement_window(sup_indipendency_win) 

795 

796 min_time_next_move = self.end_date 

797 

798 #for every agent 

799 #1. Select the Action it will execute (Action Selection phase) 

800 #2. Select the destination of its next displacement (Location Selection phase) 

801 #3. If the agent cannot move at any location, the action is corrected (Action Correction phase) 

802 

803 for agent in range(self.n_agents): 

804 

805 location_id = None 

806 

807 #if the user is spending its visiting time do nothing  

808 if self.current_date != self.agents[agent]['time_next_move']: 

809 if self.agents[agent]['time_next_move'] < min_time_next_move: 

810 min_time_next_move = self.agents[agent]['time_next_move'] 

811 continue 

812 

813 #check the current abstract location, if it is 0 i can skip all the 

814 #location selection phase and return at the Home Location, otherwise  

815 #the abstract location is mapped to a physical one through the standard procedure  

816 abstract_location = self.get_current_abstract_location_from_diary(agent) 

817 

818 #home location 

819 if abstract_location == 0: 

820 location_id = self.agents[agent]['home_location'] 

821 

822 if location_id is None: 

823 #compute p_exp, the probability that the agent will explore a new location  

824 p_exp = self.agents[agent]['rho'] * (self.agents[agent]['S'] ** -self.agents[agent]['gamma']) 

825 

826 #generate a random number for the choice: Explore or Return respectively with probability pS^-gamma and 1-pS^-gamma  

827 p_rand_exp = numpy.random.rand() 

828 

829 #generate a random number for the social or solo choice (alpha, 1-alpha) 

830 p_rand_soc = numpy.random.rand() 

831 

832 p_action = '' 

833 else: 

834 p_action = 'home_return' 

835 

836 

837 # ACTION CORRECTION PHASE 

838 if p_action == '': 

839 # compute which action the agent will execute 

840 if p_rand_exp < p_exp: 

841 if p_rand_soc < self.agents[agent]['alpha']: 

842 choice = 'social_exploration' 

843 else: 

844 choice = 'individual_exploration' 

845 else: 

846 if p_rand_soc < self.agents[agent]['alpha']: 

847 choice = 'social_return' 

848 else: 

849 choice = 'individual_return' 

850 else: 

851 choice = p_action 

852 

853 # LOCATION SELECTION PHASE 

854 if choice == 'social_exploration': 

855 location_id = self.make_social_action(agent, 'exploration') 

856 

857 elif choice == 'individual_exploration': 

858 location_id = self.make_individual_exploration_action(agent) 

859 

860 elif choice == 'social_return': 

861 location_id = self.make_social_action(agent, 'return') 

862 

863 elif choice == 'individual_return': 

864 location_id = self.make_individual_return_action(agent) 

865 

866 #ACTION CORRECTION PHASE 

867 # -1 means no selectable location 

868 

869 corrections=None 

870 if location_id == -1: 

871 location_id, corrections = self.action_correction_diary(agent, choice) 

872 

873 if location_id >= 0: 

874 info_move = self.confirm_action(agent, location_id) 

875 

876 if self.log_file is not None: 

877 logging.info("Agent "+str(agent)) 

878 logging.info("Moved from loc. "+str(info_move['from'])+" to loc. " 

879 +str(info_move['to'])+" at timestamp " 

880 +str(self.current_date)) 

881 logging.info("Action: "+choice) 

882 if corrections is None: 

883 logging.info("Corrections: None") 

884 else: 

885 str_corr = choice 

886 for corr in corrections: 

887 str_corr+=" -> "+corr 

888 logging.info("Corrections: "+str_corr) 

889 logging.info("Next move: "+str(info_move['next_move'])+"\n") 

890 

891 if self.verbose>1: 

892 print("Agent "+str(agent)) 

893 print("Moved from loc. "+str(info_move['from'])+" to loc. " 

894 +str(info_move['to'])+" at timestamp " 

895 +str(self.current_date)) 

896 print("Action: "+choice) 

897 if corrections is None: 

898 print("Corrections: None") 

899 else: 

900 str_corr = choice 

901 for corr in corrections: 

902 str_corr+=" -> "+corr 

903 print("Corrections: "+str_corr) 

904 print("Next move: "+str(info_move['next_move'])+"\n") 

905 else: 

906 #this should never happen, since n_loc>2 

907 raise Exception("Fatal error, unable to correct the location") 

908 

909 if self.agents[agent]['time_next_move']< min_time_next_move: 

910 min_time_next_move = self.agents[agent]['time_next_move'] 

911 

912 

913 self.current_date = min_time_next_move 

914 

915 if show_progress: 

916 dT2 = self.current_date - last_t 

917 if(dT2.components[0]!=0 or dT2.components[1]!=0): 

918 pbar.update(dT2.components[0]*24 + dT2.components[1]) 

919 last_t = self.current_date 

920 elapsed_h += dT2.components[0]*24 + dT2.components[1] 

921 

922 if show_progress: 

923 pbar.update(self.total_h - elapsed_h) 

924 pbar.close() 

925 

926 if self.log_file is not None: 

927 logging.shutdown() 

928 

929 self.update_agent_movement_window(self.end_date) 

930 tdf = TrajDataFrame(self.trajectories, user_id=0, latitude=1, longitude=2, datetime=3) 

931 tdf = tdf.sort_by_uid_and_datetime() 

932 return tdf
